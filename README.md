This repository contains a collection of small C programming projects designed for practice and learning. Each project focuses on fundamental programming concepts such as arrays, strings, matrices, arithmetic operations, and basic input/output handling.


The code is structured to be simple, easy to understand, and suitable for beginners who want to strengthen their problem-solving and programming skills in C.


Each project includes the source `.c` file. You can compile and run each program using any standard C compiler on macOS, Linux, or Windows. This repository serves as a handy reference for learning core C concepts and as a starting point for building more complex projects.
